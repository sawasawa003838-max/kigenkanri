# FoodStock

## 概要
FoodStock は、食品の賞味期限や消費期限を管理するための Web アプリです。  
食品名・カテゴリ・期限日を登録して、期限が近い食品や期限切れの食品を一覧で確認できます。

## できること
- 食品名、カテゴリ、期限日の登録
- 開封日と保存日数から期限日を自動計算
- 登録した食品の一覧表示
- 期限切れ、今日まで、3日以内、1週間以内で絞り込み
- 食品名での検索
- 登録内容の編集と削除
- LocalStorage を使ったブラウザ内保存

## 使用技術
- HTML
- CSS
- JavaScript
- LocalStorage
