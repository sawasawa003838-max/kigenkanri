# 賞味期限管理アプリ

## 概要
食品の賞味期限を管理するためのWebアプリです。  
食品名と期限を入力すると一覧に追加され、期限が近いものや期限切れのものを色分けして表示できます。

## 作成した理由
アルバイトで食品の期限管理の見落としが課題になっていた経験から、  
期限管理をわかりやすくする仕組みを自分で作ってみたいと考え、このアプリを制作しました。

## 主な機能
- 食品名と賞味期限の登録
- 期限が近い食品の色分け表示
- 期限切れ食品の色分け表示
- 残り日数の表示
- 不要な食品の削除
- LocalStorageを用いたデータ保存

## 工夫した点
- 一覧を賞味期限が近い順に並べるようにした
- 期限切れ、今日まで、3日以内を見た目で分かるようにした
- ページを更新してもデータが消えないようにした

## 使用技術
- HTML
- CSS
- JavaScript
- LocalStorage
